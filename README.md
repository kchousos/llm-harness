This project uses LLMs to automatically generate fuzzing harnesses for your
C/C++ project.

## Installation

### Dependencies

- Python >=3.10
- [uv](https://docs.astral.sh/uv/)

    ```
    pipx install uv
    ```

### Installation Steps

1. Clone the repository:

    ```
    git clone https://github.com/kchousos/llm-harness.git
    ```

2. Install the project:

    ```
    uv sync
    ```
